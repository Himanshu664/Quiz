## Installation
Open your terminal and then type
```bash
git clone {the url to the GitHub repo}
```
cd into the new folder and type
```bash
cd Quiz
```
Install my-project with npm

```bash
npm install
```

To run the crypto-tracker project
```bash
npm start
```
